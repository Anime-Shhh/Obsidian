### TechWithTim Greedy solutions

**Greedy solutions should be used under 2 properties:**

+ <u>Greedy Choice Property</u>: "A global Optimal solution can reached by using the optimal solution at each step"
+ <u>Optimal Substructure</u>: A problem has an optimal substructure if the optimal solutions for the whole problem contains optimal solutions to sub-problems
