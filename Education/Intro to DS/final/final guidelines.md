**Final Project Guidelines**

Your final project paper **must be 8 pages** (with unlimited reference pages) and follow the [**NeurIPS LaTeX template** Links to an external site.](https://www.overleaf.com/latex/templates/formatting-instructions-for-neurips-2026/bjdwqfdkyftc?gad_source=1&gad_campaignid=23369341867&gbraid=0AAAAAqfJXW7y5H34oKm0vAuhLECz_VmUw&gclid=Cj0KCQjwy_fOBhC6ARIsAHKFB7__2kFZPyqJM6-UWCGQx5yq8CrMeeNtM1UiM26cZOa3_hGGWg81QrAaAlXcEALw_wcB). The paper should include all standard research components, such as:

- Abstract
    
- Related Work
    
- Method
    
- Evaluation
    
- Discussion
    
- Limitations
    

Please ensure the following requirements are met:

1. **Author Information**  
    Each group must list all **student names, section ID, and NetIDs** in the author block of the final paper. Each member of the group should **submit the same proposal PDF on Canvas**.
    
2. **Code Submission**  
    Include a **public GitHub link** to your project code so the grader can access it. Add this link at the **end of the abstract**.
    
3. **Video Presentation**  
    Each group must record a **video (max 15 minutes)** using Zoom.
    
    - **All group members must appear in the video.**
        
    - Upload the video to the cloud (Zoom or other platform).
        
    - Include the **video link** at the **end of the abstract**, along with your GitHub link. Make sure all links are public available.
        
    
    Example format in your paper:
    
    ```latex
    \usepackage{hyperref}\section*{Abstract}
    Your abstract text goes here.  
    Code: \url{https://github.com/your_repo}  
    Video: \url{https://your_video_link}
    ```
